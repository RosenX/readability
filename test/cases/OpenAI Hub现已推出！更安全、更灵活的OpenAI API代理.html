OpenAI Hub现已推出！更安全、更灵活的OpenAI API代理

lightsing 发表于 2023-06-12 16:02

Tags：openai,gpt,llm

开源项目OpenAI Hub已经正式推出！OpenAI Hub是一个为OpenAI API用户提供的代理服务器，旨在提供更安全、更灵活的API使用方式。项目是用Rust编写的，我们欢迎所有有兴趣的开发者加入我们。

OpenAI Hub的主要功能包括：

  1. 负载均衡：有效利用多个API密钥，提升并发能力，方便多个用户访问同一个模型。
  2. API密钥保护：允许用户在不需要个人OpenAI API密钥的情况下发出请求，增强了安全性和易用性。
  3. 全局访问控制：调节用户对特定API和模型的访问，确保合适的人有权访问合适的资源。
  4. JWT身份验证：使用JSON Web Tokens (JWT)的安全可靠的用户身份验证系统。
  5. 访问日志：通过我们新实施的访问日志功能跟踪API的使用和令牌消耗。你可以选择在文件，SQLite，MySQL或PostgreSQL后端存储日志。

你可以在GitHub上找到我们的项目源码和详细文档：https://github.com/lightsing/openai-hub

如果你对这个项目感兴趣，我们非常欢迎你的加入！无论是使用我们的项目，还是为项目做出贡献，你的支持和反馈都是我们最大的动力。
